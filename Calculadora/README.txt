Este script te permite hacer una serie de cálculos haciendo uso de los siguientes comandos.
> Puedes añadir -i como primer argumento para que todos los cálculos se hagan de forma desasistida. Si no se pone nada se empleará el método asistido.

# ASISTIDO #
> Para hacer sumas ./calculadora.sh -s
> Para hacer restas ./calculadora.sh -r
> Para hacer multiplicaciones ./calculadora.sh -m
> Para hacer divisiones ./calculadora.sh -d
> Para hacer potencias ./calculadora.sh -e
> Para sacar el resto ./calculadora.sh -o
> Para pasar un número a binario ./calculadora.sh -b
> Para pasar un número a decimal ./calculadora.sh -l

# DESASISTIDO #
> Para hacer sumas ./calculadora.sh -i -s [Lista de operandos que se van a sumar]
> Para hacer restas ./calculadora.sh -i -r [Lista de operandos que se van a restar] *Todos los operandos se restarán al primero de la lista.
> Para hacer multiplicaciones ./calculadora.sh -i -m [Lista de operandos que se van a multiplicar]
> Para hacer divisiones ./calculadora.sh -i -d [Lista de operandos que se van a dividir]
> Para hacer potencias ./calculadora.sh -i -e [Lista de operandos que se van a potenciar]
> Para sacar el resto ./calcladora.sh -i -o [Lista de operandos de los que se va a sacar el resto]
